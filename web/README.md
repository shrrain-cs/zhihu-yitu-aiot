# Web 端代码说明

本目录用于存放“智护忆途”项目的 Web 端代码，主要面向家属端远程监护平台的前端页面开发。

当前 Web 端处于原型开发阶段，主要用于验证终端数据接入、设备状态展示、风险等级判断、电子围栏判断和监护数据记录等核心功能。后续将逐步接入 ESP32 实时上传数据、地图 API、云数据库和消息推送等功能。

## 目录结构

```text
web/
├── README.md
└── monitor-platform/
    ├── index.html
    ├── style.css
    └── script.js
```

## 文件说明

```text
index.html   页面结构文件，负责展示设备状态、数据接入表单和监护记录表格
style.css    页面样式文件，负责整体布局、颜色、卡片、表格和按钮样式
script.js    页面逻辑文件，负责数据接收、风险判断、电子围栏判断和本地数据保存
```

## 当前功能

* 终端数据接入模拟
* 设备编号、电量、运行状态展示
* 经纬度、温湿度数据显示
* 监护数据记录表格展示
* 疑似跌倒、主动求助、设备离线、长时间静止等风险判断
* 低电量告警判断
* 简单电子围栏越界判断
* 使用浏览器本地存储保存测试数据


## 当前说明

当前页面中的数据接入功能为前端模拟接收终端数据，用于项目早期功能验证。后续接入 ESP32 后，可将该部分替换为真实设备通过 HTTP、MQTT 或云平台接口上传的数据。

## 后续计划

* 接入 ESP32 真实设备数据上传
* 接入地图 API，显示实时位置
* 增加历史轨迹回放功能
* 接入云数据库，实现多端数据同步
* 增加多设备管理功能
* 增加家属端登录和设备绑定功能
* 增加告警消息推送功能

## 注意事项

本阶段数据均为模拟测试数据，不包含真实患者隐私信息。后续开发中不得将真实老人姓名、家庭住址、手机号、实时定位轨迹、WiFi 密码、数据库密钥或 API Key 上传到公开仓库。
